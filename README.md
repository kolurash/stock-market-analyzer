# 📈 Stock Market Analyzer

A Streamlit app that helps investors and students visualize stock trends, understand technical indicators, and get simple Invest/Exit/Wait suggestions.

## Features
- Fetches live data from Yahoo Finance
- Shows historical stock data (Open, High, Low, Close, Volume)
- Closing price chart
- Moving averages (20-day and 50-day)
- Decision logic for Invest/Exit/Wait

## Supported Tickers
Examples:
- AAPL → Apple
- TSLA → Tesla
- RELIANCE.NS → Reliance Industries (India)
- TATAMOTORS.NS → Tata Motors (India)
- ^NSEI → Nifty 50 Index
- ^BSESN → Sensex Index
